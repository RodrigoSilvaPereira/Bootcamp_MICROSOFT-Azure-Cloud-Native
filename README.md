# ☁️ Bootcamp Microsoft Azure Cloud-Native

Este repositório reúne toda a jornada de aprendizado e prática em **Microsoft Azure**, abordando desde os **fundamentos de computação em nuvem** até a construção de uma aplicação **cloud-native completa, escalável e orientada a eventos**.

O conteúdo está estruturado em módulos progressivos, combinando:

- fundamentos teóricos
- laboratórios práticos
- construção de arquiteturas reais

---

# 🚀 Objetivo do Repositório

O objetivo deste repositório é consolidar conhecimentos em:

- computação em nuvem
- arquitetura de sistemas distribuídos
- desenvolvimento cloud-native
- microsserviços
- serverless
- DevOps e boas práticas

Ao final, é construída uma aplicação completa utilizando múltiplos serviços da Azure.

---

# 🧠 Tecnologias e Serviços Utilizados

Este repositório utiliza diversos serviços da Azure, incluindo:

- Azure App Service

# 🧩 Arquitetura do Projeto Final

A aplicação final utiliza:

- Frontend + BFF
- Azure Functions (processamento)
- Service Bus (mensageria)
- PostgreSQL (dados transacionais)
- Cosmos DB (status e eventos)
- Logic Apps (notificações)
- Key Vault (segurança)
- Application Insights + Monitor (observabilidade)

---

# 🔄 Padrões Arquiteturais Aplicados

Durante o bootcamp foram aplicados padrões modernos:

| Padrão | Aplicação |
|------|--------|
| Microsserviços | separação de domínios |
| Serverless | processamento sob demanda |
| Event-driven | comunicação assíncrona |
| BFF | otimização do frontend |
| Multi-database | banco ideal para cada contexto |
| Observabilidade | monitoramento completo |
