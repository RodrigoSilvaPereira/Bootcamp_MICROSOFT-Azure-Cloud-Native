# ☁️ Fundamentos da Plataforma Azure

Este diretório reúne conteúdos introdutórios sobre a **plataforma Microsoft Azure**, abordando conceitos fundamentais de computação em nuvem, arquitetura da plataforma e exemplos práticos de uso dos serviços.

O objetivo deste módulo é compreender **como a infraestrutura da Azure funciona** e como seus serviços podem ser utilizados para construir aplicações modernas na nuvem.

---

# 📂 Estrutura do Módulo

Este módulo está dividido em duas partes principais: **conteúdo conceitual** e **desafio prático**.

## 📘 1 - Aplicações Azure

```
1-Aplicacoes-Azure/
```

Contém a documentação dos **conceitos fundamentais da plataforma Azure**, incluindo:

- Computação em nuvem
- Modelos de serviço (IaaS, PaaS e SaaS)
- Arquitetura da Azure
- Regiões e zonas de disponibilidade
- Serviços de computação
- Serviços de armazenamento

Esse material serve como **base teórica para compreender o funcionamento da plataforma**.

---

## 🧪 2 - Desafio: Armazenando Dados de um E-Commerce

```
2-Desafio/
```

Contém um **laboratório prático** onde foi construída uma solução de armazenamento em nuvem utilizando serviços da Azure.

A aplicação utiliza:

- **Azure SQL Database** para armazenamento de dados estruturados
- **Azure Blob Storage** para armazenamento de imagens
- **Python + Streamlit** para interface da aplicação

Esse desafio demonstra na prática **como integrar serviços da Azure para construir uma aplicação real**.

# 🎯 Objetivo do Módulo

Ao final deste módulo é possível compreender:

- os fundamentos da computação em nuvem
- a arquitetura básica da plataforma Azure
- como serviços de armazenamento e banco de dados funcionam na prática
- como integrar aplicações com serviços cloud
