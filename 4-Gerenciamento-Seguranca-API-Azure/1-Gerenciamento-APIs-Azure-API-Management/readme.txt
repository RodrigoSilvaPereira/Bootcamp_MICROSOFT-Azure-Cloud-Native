Módulo: Gerenciamento de APIs com Azure API Management

API Management

O gerenciaento de API ajuda as organizações a publicar APIs para parceiros externos e desenvolvedores internos a fim de desbloquear o potencial de seus dados e serviços

O Papel do Gerenciamento de API

O gerenciamento de API fornece funcionalidades fundamentais para garantir um programa de API bem-sucedido com base no envolvimento do desenvolvedor, insights de negócios, análise, segurança e proteção.
Cada API consiste em uma ou mais operações e cada uma pode ser adicionada a um ou mais produtos

O sistema é composto pelos seguintes componentes:

- Gateway de API
- Portal do Azure
- O portal do Desenvolvedor

A divisão é feita entre:

Produtos - Os produtos são como as APIs são exibidas para os desenvolvedores
Grupos - Usados para gerenciar a visibilidade dos produtos para os devs
Desenvolvedores - Representma as contas de usuários em uam instância de serviço de gerenciamento de API
Políticas - Permite que o portal do azure use conffigurações para alterrar o comportamente da API
Portal do Desenvolvedor - Você pode aprener sobre sua API, ver e chamar operações e assinar produtos

Função do Gateway de API

- Fica entre cliente e serviços
- Atua como proxy reverso
- Aplica políticas e coleta telemetria
- Autenticação, terminação SSL e limitação de taxa
