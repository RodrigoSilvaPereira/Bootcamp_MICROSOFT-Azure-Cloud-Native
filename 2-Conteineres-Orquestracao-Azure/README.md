# ☁️ Conteineres e Orquestração na Azure

Este diretório reúne conteúdos relacionados à **execução, hospedagem e orquestração de aplicações na plataforma Microsoft Azure**.

Neste módulo são exploradas duas abordagens importantes utilizadas no desenvolvimento de aplicações modernas na nuvem:

- **orquestração de contêineres com Kubernetes**
- **hospedagem de aplicações web utilizando serviços PaaS**

Esses serviços permitem construir aplicações **escaláveis, resilientes e cloud-native**, utilizando os recursos gerenciados da plataforma Azure.

---

# 📂 Estrutura do Módulo

Este módulo está dividido em duas partes principais, cada uma abordando uma estratégia diferente para execução de aplicações na nuvem.

---

# ☸️ 1 - Azure Kubernetes Service

```
1-Azure-Kubernetes-Service/
```

Contém a documentação sobre **orquestração de contêineres utilizando Kubernetes na Azure**.

Entre os conceitos abordados estão:

- Azure Kubernetes Service (AKS)
- Azure Container Registry (ACR)
- criação de imagens Docker
- gerenciamento de imagens de contêiner
- conceitos fundamentais do Kubernetes
- deploy de aplicações containerizadas

Esse conteúdo demonstra como construir e executar **aplicações baseadas em contêineres utilizando Kubernetes na plataforma Azure**.

---

# 🌐 2 - Azure App Service

```
2-Azure-App-Service/
```

Contém a documentação sobre **hospedagem de aplicações web utilizando Azure App Service**.

Entre os conceitos abordados estão:

- criação de Web Apps
- hospedagem de APIs e aplicações web
- planos do App Service
- dimensionamento automático
- autenticação integrada
- integração com pipelines de deploy

Esse serviço permite executar aplicações **sem necessidade de gerenciar servidores ou infraestrutura**, utilizando o modelo **Platform as a Service (PaaS)**.

---

# 🎯 Objetivo do Módulo

Ao final deste módulo é possível compreender duas abordagens comuns para execução de aplicações na nuvem:

### 🔹 Arquitetura baseada em contêineres

Utilizando:

- Docker
- Azure Container Registry
- Azure Kubernetes Service

Essa abordagem é ideal para **arquiteturas distribuídas e microsserviços**.

---

### 🔹 Hospedagem gerenciada de aplicações

Utilizando:

- Azure App Service

Essa abordagem é ideal para **aplicações web e APIs que precisam de implantação rápida e gerenciamento simplificado**.
